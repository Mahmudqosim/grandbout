# Welcome to Grandbout!

Grandbout is a canvas based fighting game with 6 characters (still gonna add more).

## Usage

- Clone the repository
- Install the dependencies:

```bash
npm i
```

- Once finished, you can start the development server:

```
npm run dev
```
